WEB DEVELOPMENT SOFTWARE INSTALLATION

v1.0.4

The script included in this repo installs Mac-related configuration files and software required for any web development course you take with me. Before continuing, open a GitHub account at https://github.com/, regardless of whether you use Windows, Mac OS, or Linux.

Mac OS Users

  1. Runs Software Update
  2. Installs the Command Line Tools
  3. Enables Mac OS’s ability to cycle through IP address, hostname, and OS version when clicking the clock in the login window
  4. Enables UTF-8 input into The Terminal
  5. Installs the Homebrew package manager
  6. Installs the IBM Plex Mono typeface
  7. Installs the Ubuntu Mono typeface
  8. Installs .editorconfig
  9. Installs .stylelintrc.json
  10. Installs .git-prompt.sh
  11. Installs .git-completion.sh
  12. Installs .bash_aliases
  13. Installs .bashrc
  14. Installs .bash_profile
  15. Optionally installs .sass-lint.yml
  16. Optionally installs .eslintrc.json
  17. Installs the Tomorrow Night theme for The Terminal
  18. Installs Rectangle
  19. Installs VS Code
  20. Installs GitHub Desktop

Installation

The enclosed script installs Mac-based HTML-, CSS-, Sass-, and JavaScript-related software and configuration files for any web development course you take with me. If you’re using macOS Catalina, Big Sur, or Monterey, then you’ll first need to grant The Terminal access to your disk. Follow this tutorial, then proceed below.

1 / Open The Terminal

The Terminal is called Terminal.app and is in /Applications/Utilities. Alternatively, you can perform a Spotlight search for “Terminal”.

2 / Navigate to The Root of This Repo

In The Terminal, type cd followed by a space. Locate the install-web-dev-software folder (this repo). Click the folder icon in the top of the tab bar and drag it into The Terminal’s window. Hit return.

3 / Grant the Script Access

Give the script executable access to your file system by typing the following: chmod 755 install.sh

4 / Run The Installer

Type ./install.sh and then hit return.

Update bash to v5

Source: Upgrading Bash on macOS by Daniel Weibel

  1. Launch The Terminal
  2. Run brew install bash
  3. Edit the shells file using Atom: sudo atom /etc/shells
  4. Add /usr/local/bin/bash to the tail of the shells file
  5. Set the default shell for the current user: chsh -s /usr/local/bin/bash
