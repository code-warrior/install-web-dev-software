SOFTWARE INSTALLATION FOR WEB DESIGN COURSE AT RISD

The script included in this repo installs Mac-related configuration files and software required for my Web Design course at RISD. Before continuing, open a GitHub account at https://github.com/, regardless of whether you use Windows, Mac OS, or Linux.

Mac OS Users

What This Script Does to Your Mac

  1. Runs Software Update
  2. Installs the Command Line Tools
  3. Enables Mac OS’s ability to cycle through IP address, hostname, and OS version when clicking the clock in the login window
  4. Enables UTF-8 input into The Terminal
  5. Installs the FTP program FileZilla
  6. Installs the Typora Markdown reader/viewer
  7. Installs the IBM Plex Mono typeface
  8. Installs the Ubuntu Mono typeface
  9. Installs .editorconfig
  10. Installs .stylelintrc.json
  11. Installs .git-prompt.sh
  12. Installs .git-completion.sh
  13. Installs .bash_aliases
  14. Installs .bashrc
  15. Installs .bash_profile
  16. Installs the Tomorrow Night theme for The Terminal
  17. Installs Spectacle
  18. Installs and configures Atom
  19. Installs GitHub Desktop

Installation

1 / Open The Terminal

The Terminal is called Terminal.app and is in /Applications/Utilities. Alternatively, you can perform a Spotlight search for “Terminal”.

2 / Navigate to The Root of This Repo

In The Terminal, type cd followed by a space. Locate the install-web-dev-software folder (this repo). Click the folder icon in the top of the tab bar and drag it into The Terminal’s window. Hit return.

3 / Run The Installer

Type ./install.sh and then hit return.

Windows and Linux Users

All the software you need must be downloaded/installed manually or via a different script. Visit the Software Requirements section of the syllabus for more.
